BBMEnterprise.Event.ChatTyping

The structure of the BBMEnterprise.Messenger#event:chatTyping event. This event is emitted when a participant has started or stopped typing within a chat.

Multiple BBMEnterprise.Messenger#event:chatTyping events having an 'isTyping' value of true may be emitted without an intervening event that has an 'isTyping' value of false. The identified user is either typing or not within the chat. The 'tag' and 'messageId' fields provide only the current context about user that is typing and do not represent different overlapping typing states for the user.
