BBMEnterprise.Messenger.AppMessage

This class represents an app message received from the Spark infrastructure. App messages are app-specific messages with app-defined content sent to one or more identities for delivery to all their registered endpoints. App messages can be sent using the Spark REST API.

No references to this object or its properties are retained by the SDK.

The data of the app message in the form of a JSON object. The meaning of the JSON object's contents are defined by the app and are opaque to the SDK.
